# Sample Company Folder

This is a placeholder showing the expected folder structure. Replace it with your actual company folders.

## How to use

1. Delete this sample folder
2. Create one folder per portfolio company: `{Sector} - {Company Name}/`
3. Drop update files (PDF, Excel, PPT) into the folder

## Example folder names

```
Fintech - Acme Payments/
Climate - GreenCo/
Health - MedTech/
SaaS - DataFlow/
Consumer - FreshBites/
```

## What files to drop in

- Quarterly board decks (PDF/PPT)
- MIS / financial statements (Excel)
- Investor update emails (PDF)
- KPI dashboards (Excel)
- Any document with company financials or operational data
